# Archived Changesets

This folder keeps already-shipped changeset files out of [`.changeset/`](../../../.changeset/).

Changesets only tracks pending releases. Once a release has shipped, leave the generated changelog entries in place and move the source changeset files here so future `changeset status` and `changeset version` runs only see unreleased work.

The `2026-03-v4-initial-release/` batch contains the initial V4 launch changesets that shipped with:

- `codegraphy@4.0.0`
- `codegraphy-typescript@1.0.0`
- `codegraphy-python@1.0.0`
- `codegraphy-csharp@1.0.0`
- `codegraphy-godot@1.0.0`
- `@codegraphy-vscode/plugin-api@0.1.0`
