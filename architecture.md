# Architecture

Living repo map for tracked files. Read at conversation start. Update before commits.

## Boundaries

- Root config/docs: workspace tooling, release notes, contributor workflow, project plans.
- `packages/extension`: VS Code extension host, shared types, React webview, extension tests.
- `packages/plugin-api`: plugin contracts for extension/webview/plugin authors.
- `packages/plugin-*`: built-in language plugins with examples and focused tests.
- Generated or local-only artifacts are intentionally omitted: `node_modules`, `dist`, `coverage`, `reports`, `.stryker-tmp`, package-local caches.
