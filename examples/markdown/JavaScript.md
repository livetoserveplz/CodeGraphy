# JavaScript

The language of the web. Dynamically typed, prototype-based.

## Related

- [[TypeScript]] — adds static types on top of JS
- [[React]] — popular UI library built with JS
- [[Tooling]] — bundlers and package managers

## Notes

JavaScript runs in both browsers and [[Node]] environments.
Modules use `import`/`export` (ESM) or `require` (CJS).
