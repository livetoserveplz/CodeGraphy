# Tooling

Build tools and developer setup for [[JavaScript]] and [[TypeScript]] projects.

## Bundlers

- **esbuild** — ultra-fast bundler, used for the extension host
- **Vite** — dev server + bundler, used for the [[React]] webview

## Package Managers

- npm / pnpm / bun

## Linting & Formatting

- ESLint — static analysis
- Prettier — code formatting
