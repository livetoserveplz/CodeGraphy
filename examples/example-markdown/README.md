# Example Markdown

Small Obsidian-style workspace for testing wildcard Markdown wikilink parsing.

## Symbol Node Demo

Suggested symbol check:

1. Open `notes/Home.md`.
2. In Graph Scope, enable **Symbol**.
3. Search for `parseCommentedLink` from `src/commented.ts`.

Expected behavior:

- Markdown reference edges connect the notes and the TypeScript file.
- The TypeScript symbol gives the mixed docs/code graph a concrete code declaration to navigate to.
